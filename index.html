What can be found here?

There are all the current available tutorials provided in sequential order. Please at least look over each lesson to confirm you understand the material found within before moving along.

Downloads: You can download the entire directory in a .tar or a .zip format.
Files come in a .ipynb (Jupyter Notebook) format.

To open the files, first install Jupyter Notebook with: 
pip3 install jupyter

Then, in the uncompressed downloaded repo directory, initialize the notebook: 
jupyter notebook
